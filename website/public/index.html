What is KubeVirt

KubeVirt is an open source project that allows Virtual Machines (VMs) to be run and managed as pods inside a Kubernetes cluster.

What is our Objective?

We are demonstarting how kubevirt is orchestrating both container pods and VM, and presenting it as single application to the outside wrold.
